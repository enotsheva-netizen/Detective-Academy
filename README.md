# Detective Academy

**Case File 017: The Blackwood Manor Murder** is an offline, interactive B1+ speaking lesson on crimes, punishment, investigation vocabulary, and modals of deduction.

## Open the lesson

Double-click `index.html` or drag it into a modern browser. No installation, account, internet connection, or local server is required.

## Included lesson stages

1. Case briefing and warm-up speculation
2. Vocabulary cards and a quick check
3. Grammar discovery and controlled practice: `must have`, `might have`, `can't have`
4. Interactive crime-scene evidence
5. Suspect profiles and interrogations
6. Evidence board and jury verdict
7. Sealed final reveal
8. Teacher mode with timings, answers, solution, and notes

## Teacher use

Use **Teacher mode** in the top-right corner to access the answer guide. Keep the final reveal closed until students have submitted a verdict.

## Customising the case

- The active lesson content and interactions are in `script.js`.
- Visual design is in `style.css`.
- `data/case017.json` is a compact case record for future content editing. The site also embeds the current content so it works when opened directly from a computer.
- `assets/placeholders/` is ready for optional portraits, evidence images, or audio.
